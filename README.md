A robust end-to-end (E2E) testing framework designed for the Task Manager Web application, built with Playwright and TypeScript.
🚀 Overview

This project focuses on automating critical user workflows for a task management application. By utilizing Playwright, the test suite ensures high reliability and fast feedback loops, simulating real-world user interactions across different browsers.
🛠 Tech Stack

    Language: TypeScript

    Testing Framework: Playwright

    Environment: Node.js

📋 Features

    E2E Testing: Automated CRUD workflows for task management.

    Cross-browser support: Testing capabilities for Chromium, Firefox, and WebKit.

    CI/CD Ready: Configured with GitHub Actions for automated test runs on every push.

    Report Generation: Generates detailed HTML reports for every test execution.

⚙️ Getting Started
Prerequisites

Make sure you have Node.js installed on your machine.
Installation

    Clone this repository:
    Bash

    git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git

    Install the dependencies:
    Bash

    npm install

Running Tests

To run the tests in headless mode:
Bash

npx playwright test

To view the latest test report:
Bash

npx playwright show-report
